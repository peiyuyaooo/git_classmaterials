新增code之後 or any changes in VScode，左側的source control就會出現變更數字。
source control 中的 message = 註記當次修改的內容 (notes)
輸入完畢之後點擊 commit 送出並將紀錄儲存在 暫存空間 ../vscode.git。
or you can type shortkey in terminal like:
1. git add
2. git commit -m "message"

( ⌘ + ⇧ + . ) 顯示暫存檔案

how to push / create new repository on github

1. add new repositories at the left side of github.com 2. enter the Repository name (english only) 3. back to your VScode and ensure you commit all the changes 4. push the changes to the new repository 5. copy the command below the page on github.com and paste to your terminal
